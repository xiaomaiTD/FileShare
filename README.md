# FileShare
## wifi下通过PC实现文件共享，方便设备浏览和编辑多媒体文件，支持以下媒体文件。
 * 视频文件：rmvb，mp4，3gp等。
 * 图片文件：png，jpep等。
 * 支持iOS平台下的办公文件，比如xlsx文件
 * 支持pdf，text等文件
